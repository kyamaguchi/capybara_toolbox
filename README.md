## Capybara Toolbox

Demo app with capybara tips

### Running spec

```
rspec
```

Running spec with chrome

```
CHROME=true rspec
```

Use guard to run spec

```
guard
```

### Tips

- General
  - Confirmation Dialog
- jQueryUI
  - Sortable
  - Resizable
